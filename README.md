# Weather Dashboard

## Purpose

This app provides daily weather information as well as a 5 day forecast.
The information is retrieved from https://openweathermap.org using API data.
The information displyed in the app is as follows:

  - Temperature
  - Wind speed
  - Humidity
  - UV Index

* CSS
* BOOTSTRAP
